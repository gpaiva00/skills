# CLAUDE.md

This is the root of the skills repository. Each skill lives in its own directory under `skills/<category>/<skill-name>/` and contains a `SKILL.md` file.

## Repository Structure

```
skills/
  <category>/
    <skill-name>/
      SKILL.md
      README.md (optional)
      ...
```

## Adding a New Skill

1. Create a directory under `skills/<category>/<skill-name>/`
2. Add a `SKILL.md` with the skill definition
3. Update this README.md with the skill reference
4. Commit and push
